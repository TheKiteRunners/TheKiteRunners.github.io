mysql Community Server 5.7安装教程以及常见错误

发表于 2017-09-24 | 分类于 数据库 | | 本文总阅读量 次

(如果之前有安装过mysql, 建议找个卸载文章, 先把mysql删除干净)

步骤

  1. 从官网下载完之后https://dev.mysql.com/downloads/mysql/解压文件
  2. 找到文件夹下的bin设置环境变量.
  3. (…..这时候如果直接运行mysql就会提示”Can’t connect to MySQL server on localhost (10061)”… 坑爹.) 使用管理员权限进入cmd命令提示符, 使用cd命令进入到mysql的bin文件夹下.
  4. 输入命令
    D:\MySQL\bin> mysqld --install Service successfully installed.
  5. 初始化mysql数据库，输入
    mysqld --initialize --user=root --console 。

    最下方会输出类似的内容:
    A temporary password is generated for root@localhost: u).ig.Gd.7hW

    root@localhost: 后面是数据库自动生成的密码.
  6. 输入命令: “net start mysql”

    注: 测试机器: win10, 必须使用管理员权限运行cmd输入命令.
  7. 使用生成的密码登录mysql后，再里面通过set password=password('123456')修改密码。此处可以将root密码设置为123456
    mysql> set password=password('123456'); Query OK, 0 rows affected, 1 warning (0.00 sec)
  8. 由于在官网下载的是ZIP Archive版本, 没有my.ini文件甚至default的我都没找到…..这就意味着我们需要自己配置文件, 如果不配置可能由于字符集的原因导致中文乱码.
