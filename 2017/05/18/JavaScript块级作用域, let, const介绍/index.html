let

块级作用域

ES6里增加了let，const, 声明在{}, -> “function,if ,for等” 的变量作用域被限制在块级。（严格模式下 function 变量属于块级作用域）

同时还可以解决闭包问题, 例如我们为10个div绑定点击事件, 肯定会去写一个立即执行函数去解决闭包问题

但在es6中就可以使用let解决这一问题

块级作用域特点

一旦let在作用域中声明变量 就将统治这个作用域

因此使用let无法重复定义变量.

ES6 let 不存在变量声明提升

const

特点一: 一旦声明常量的值就不能更改

特点二: 声明一个常量必须马上初始化

特点三: let 和 const 一样不能重复声明
