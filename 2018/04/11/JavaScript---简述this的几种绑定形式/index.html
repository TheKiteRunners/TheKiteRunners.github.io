new绑定

使用new来调用函数, 或者说发生构造函数调用时, 会自动执行下面的操作

  1. 创建(或者构造)一个全新的对象
  2. 这个新对象会被执行 [[原型]] 连接, 也就是这个对象的 proto 指向构造函数的prototype
  3. 这个新对象会绑定到函数调用的this
  4. 如果函数没有返回其他对象, 那么new表达式中的函数调用会自动返回这个新对象
